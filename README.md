Problem Statement — SolarToAll Calculator 

Background
The adoption of rooftop solar systems is growing rapidly, but most homeowners and small businesses lack clear, simple tools to understand whether solar is financially viable for them.
________________________________________
Problem
Consumers often face the following challenges:
•	Difficulty in estimating the required solar system size 
•	Lack of clarity on installation cost 
•	Uncertainty about monthly savings and return on investment 
•	Complex or inaccessible solar calculators that require technical knowledge 
As a result, many potential users delay or avoid switching to solar energy, slowing down the transition to clean energy.
________________________________________
Objective
The goal of this project is to develop a simple and user-friendly solar calculator web app that:
•	Takes basic input like monthly electricity bill 
•	Estimates solar system size (kW) 
•	Calculates approximate installation cost 
•	Provides expected monthly savings 
________________________________________
Proposed Solution
This project presents a lightweight web application that uses real-world solar assumptions to instantly generate estimates, helping users:
•	Make informed decisions 
•	Understand cost vs savings 
•	Explore solar adoption without technical complexity 
________________________________________
Impact
•	Encourages adoption of renewable energy 
•	Simplifies solar decision-making 
•	Bridges gap between users and solar technology
